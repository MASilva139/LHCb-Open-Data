# Noticia

Este repositorio contiene material educativo adaptado del Proyecto de Datos Abiertos del LHCb y cuadernos educativos relacionados de Física sin Fronteras / Hackaton

Proyecto original:
- LHCb Open Data Project
- https://github.com/matchuf/PWF_Gt-Hn-CR_LHCb_open-data

Licencia original:
- GNU General Public License v2.0

Este repositorio reorganiza, adapta y amplía el materiall para uso didáctico, incluyendo:
- cuadernos para estudiantes,
- cuadernos resueltos,
- ejecución local/Windows sin ROOT usando UpROOT,
- ejecución local/Ubuntu usando ROOT,
- ejemplos de modularización
- utilizades gráficas adicionales,
- guías explicativas

Los datos utilizados en los ejercicios deben descargarse de CERN Open Data según las instrucciones del repostorio.
- Archivos root `B2HHH_MagnetUp.root`, `B2HHH_MagnetDown.root` y `PhaseSpaceSimulation.root` pueden encontrarse en la página de datos abiertos del [CENR](https://opendata.cern.ch/record/4900)
